# TrouserCo Advisor: Profit Optimisation & Strategy Planner

An interactive, data-driven executive dashboard designed to model and simulate e-commerce strategic levers, customer acquisition channels, and profitability outcomes. 

This application bridges the gap between raw data analysis and strategic decision-making, allowing stakeholders to dynamically stress-test growth strategies, simulate operational shifts, and instantly see the bottom-line financial impact.

## 📊 Core Business Value & Features

* **Dynamic Lever Simulation:** Real-time modeling of critical retail variables including Conversion Rate Increases, Average Order Value (AOV) Boosts, and Traffic Growth.
* **Granular Acquisition Channel Breakdown:** Pre-loaded baseline datasets mapping performance metrics across Organic Search, Paid Search, Social Media, Direct Traffic, and Email Marketing.
* **Financial Impact Modeling:** Instant calculation of complex variables such as Contribution Margin, Customer Acquisition Cost (CAC), Return on Ad Spend (ROAS), and Net Profit Adjustments.
* **Modern Tech Stack:** Engineered using **React**, **TypeScript**, **Vite**, and **Tailwind CSS** for a highly responsive, high-performance user experience.

---

## 🛠️ Data Architecture & Analytical Foundations

The core logic of the calculator processes baseline historical performance data against user-defined strategic variances. Key formulas embedded within the application infrastructure include:

* **Customer Acquisition Cost (CAC):** 
  $$\text{CAC} = \frac{\text{Total Marketing Budget}}{\text{Total Acquired Users}}$$
* **Return on Ad Spend (ROAS):** 
  $$\text{ROAS} = \frac{\text{Channel Revenue}}{\text{Ad Spend Budget}}$$
* **Net Profit Allocation:** 
  $$\text{Net Profit} = (\text{Total Revenue} \times \text{Contribution Margin \%}) - \text{Total Ad Spend} - \text{One-Time Development Budgets}$$

## 💻 Technical Setup & Local Installation

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**

git clone [https://github.com/Dariusd7/Ecommerce-strategy-and-profit-calculator.git](https://github.com/Dariusd7/Ecommerce-strategy-and-profit-calculator.git)

Navigate to the project directory:

Bash
   cd Ecommerce-strategy-and-profit-calculator
Install dependencies safely:

Bash
   npm install --legacy-peer-deps
Launch the local development server:

Bash
   npm run dev
